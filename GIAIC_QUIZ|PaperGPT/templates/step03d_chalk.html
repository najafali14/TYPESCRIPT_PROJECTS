TypeScript Quiz with Chalk Module

1. What is the output of the following TypeScript code using the Chalk module?


import chalk from 'chalk';

2. How would you print "Hello World" in red color using the Chalk module?


import chalk from 'chalk';

console.log(chalk.red('Hello World'));

3. What is the output of the following TypeScript code using the Chalk module?


import chalk from 'chalk';

let message: string = "Hello World";
console.log(chalk.yellow(message));

4. How can you underline "Hello World" using the Chalk module?


import chalk from 'chalk';

console.log(chalk.underline('Hello World'));

5. What is the output of the following TypeScript code using the Chalk module?


import chalk from 'chalk';

let message: string = "TypeScript";
console.log(chalk.bgBlue.white.bold(message));

6. Which Chalk method would you use to highlight "Hello World" in inverse colors?


import chalk from 'chalk';

console.log(chalk.inverse('Hello World'));

7. How would you combine multiple Chalk styles to print "Hello World" in bold and underlined?


import chalk from 'chalk';

console.log(chalk.bold.underline('Hello World'));

8. What is the output of the following TypeScript code using the Chalk module?


import chalk from 'chalk';

console.log(chalk.red(message) + ' ' + chalk.blue('World'));
