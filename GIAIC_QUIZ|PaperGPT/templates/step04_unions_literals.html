TypeScript Quiz: Unions and Literals

1. What is the type of the variable status in the following TypeScript code?

status = "success";
console.log(status);

status = "failure";
console.log(status);

2. What will TypeScript infer the type of result to be?

3. What is the inferred type of the variable age in the following TypeScript code?


let age: number | string;

age = 25;
console.log(age);

age = "twenty-five";
console.log(age);

4. Which of the following is a valid assignment for the variable status?


let status: "open" | "closed" | "pending";

5. What will the following TypeScript code output?


let response: "yes" | "no" = Math.random() > 0.5 ? "yes" : "no";

if (response === "yes") {
    console.log("Affirmative");
} else {
    console.log("Negative");
}
