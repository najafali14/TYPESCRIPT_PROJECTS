TypeScript Quiz: Structural Typing and Object Literals

1. What will the following TypeScript code output?


interface Ball {
    diameter: number;
}

interface Sphere {
    diameter: number;
}

let ball: Ball = { diameter: 10 };
let sphere: Sphere = { diameter: 20 };

sphere = ball;
ball = sphere;
